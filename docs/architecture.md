# Архитектура SKiss

## Обзор
SKiss - multi-tenant SaaS платформа с микросервисной архитектурой.

## Стек
- Backend: NestJS + TypeScript + PostgreSQL + Redis
- Frontend: React + Vite + TailwindCSS
- DevOps: Docker + Nginx

## Модули
1. Auth - Аутентификация и авторизация
2. User - Управление пользователями
3. Site - Управление сайтами
4. Wallet - Криптовалютный кошелек
5. Ticket - Система поддержки
6. Admin - Административная панель
